# Scripts Directory

This directory contains utility scripts for development, debugging, and data analysis.

## Categories

### Data Analysis
- `analyze_structure.py` - Analyze data structure and schema
- `check_all_files.py` - Check all data files for completeness
- `check_columns.py` - Verify column mappings
- `check_completeness_data.py` - Validate data completeness
- `check_raw_values.py` - Inspect raw data values
- `check_structure.py` - Check data structure integrity

### Debugging
- `debug_study6.py` - Debug specific study data
- `inspect_study17.py` - Inspect study 17 data
- `inspect_study17_raw.py` - Inspect raw study 17 data

### Utilities
- `list_available_models.py` - List available AI models
- `list_models.py` - List model configurations
- `quick_fix_metrics.py` - Quick metric calculation fixes
- `rename_study_folders.py` - Rename study data folders
- `verify_upload_feature.py` - Verify file upload functionality
- `wait_for_load.py` - Wait for data loading completion

## Usage

Run scripts from the project root directory:
```bash
python scripts/script_name.py
```
